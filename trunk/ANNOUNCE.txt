Announcement
------------

	Alumni-OnLine was just a group of web pages which designed for Çankaya University
alumni association (http://alumni.cankaya.edu.tr) at the begining. A couple of weeks later
it became a content management system for alumni associations. 

        With the new features that ships with the version 3.0.0.0, Alumni-OnLine is well
featured content management system for alumni associations, as well as open source community
sites! 

	Please feel free to send your comments to alumni-online@enterprisecoding.com

Description
-----------

	Alumni-OnLine is design for University alumni organization with php and DBMS support.
Since it is design with classes, it will be use MySql, MSsql, Oracle, Interbase and the other Database
systems. It can be use on any web server support php.
